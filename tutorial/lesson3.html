الدرس الثالث: إنشاء نموذج تسجيل دخول

في هذا الدرس، سنتعلم كيفية إنشاء نموذج تسجيل دخول باستخدام HTML و CSS و JavaScript، وكيفية التحقق من صحة بيانات المستخدم.

ملاحظة هامة: هذا الدرس مخصص للأغراض التعليمية فقط. يجب عدم استخدام هذه المعرفة لأغراض غير قانونية أو غير أخلاقية مثل خداع المستخدمين أو سرقة بياناتهم.

هيكل نموذج تسجيل الدخول

يتكون نموذج تسجيل الدخول عادةً من العناصر التالية:

  • حقل إدخال لاسم المستخدم أو البريد الإلكتروني
  • حقل إدخال لكلمة المرور
  • زر تسجيل الدخول
  • رابط "نسيت كلمة المرور"
  • خيار "تذكرني"
  • رابط للتسجيل إذا لم يكن لدى المستخدم حساب

لنبدأ بإنشاء هيكل HTML الأساسي لنموذج تسجيل الدخول:

إضافة التحقق من صحة البيانات باستخدام JavaScript

الآن لنضيف بعض الكود JavaScript للتحقق من صحة بيانات المستخدم قبل إرسال النموذج:

// الحصول على عناصر النموذج
const loginForm = document.getElementById('loginForm');
const usernameInput = document.getElementById('username');
const passwordInput = document.getElementById('password');

// إضافة معالج حدث لتقديم النموذج
loginForm.addEventListener('submit', function(event) {
    // منع السلوك الافتراضي للنموذج (إعادة تحميل الصفحة)
    event.preventDefault();
    
    // الحصول على قيم الإدخال
    const username = usernameInput.value.trim();
    const password = passwordInput.value.trim();
    
    // التحقق من صحة البيانات
    let isValid = true;
    let errorMessage = '';
    
    // التحقق من اسم المستخدم
    if (username === '') {
        isValid = false;
        errorMessage = 'يرجى إدخال اسم المستخدم أو البريد الإلكتروني';
    } 
    // التحقق من البريد الإلكتروني إذا كان المدخل يبدو كبريد إلكتروني
    else if (username.includes('@')) {
        const emailRegex = /^[^\s@]+@[^\s@]+\.[^\s@]+$/;
        if (!emailRegex.test(username)) {
            isValid = false;
            errorMessage = 'يرجى إدخال بريد إلكتروني صالح';
        }
    }
    
    // التحقق من كلمة المرور
    if (password === '') {
        isValid = false;
        errorMessage = 'يرجى إدخال كلمة المرور';
    } else if (password.length < 6) {
        isValid = false;
        errorMessage = 'يجب أن تتكون كلمة المرور من 6 أحرف على الأقل';
    }
    
    // عرض رسالة الخطأ أو إرسال النموذج
    if (!isValid) {
        alert(errorMessage);
    } else {
        // في تطبيق حقيقي، هنا سنرسل البيانات إلى الخادم
        alert('تم تسجيل الدخول بنجاح!');
        
        // يمكننا إعادة توجيه المستخدم إلى صفحة أخرى
        // window.location.href = 'dashboard.html';
    }
});

        // الحصول على عناصر النموذج
        const loginForm = document.getElementById('loginForm');

                isValid = false;

            // التحقق من كلمة المرور

نصائح أمان لنماذج تسجيل الدخول

عند تطوير نماذج تسجيل دخول حقيقية، يجب مراعاة الأمور التالية:

  • استخدام HTTPS: تأكد من أن موقعك يستخدم بروتوكول HTTPS لتشفير البيانات المرسلة بين المتصفح والخادم.
  • حماية من هجمات CSRF: استخدم رموز CSRF لمنع هجمات طلب التزوير عبر المواقع.
  • تقييد محاولات تسجيل الدخول: قم بتقييد عدد محاولات تسجيل الدخول الفاشلة لمنع هجمات القوة الغاشمة.
  • تخزين كلمات المرور بشكل آمن: لا تقم أبدًا بتخزين كلمات المرور كنص عادي. استخدم خوارزميات تجزئة قوية مثل bcrypt أو Argon2.
  • المصادقة الثنائية: فكر في تنفيذ المصادقة الثنائية (2FA) لطبقة إضافية من الأمان.
  • تسجيل الخروج التلقائي: قم بتسجيل خروج المستخدمين تلقائيًا بعد فترة من عدم النشاط.

تمرين عملي

حاول تحسين نموذج تسجيل الدخول بإضافة الميزات التالية:

  1. إضافة زر لإظهار/إخفاء كلمة المرور
  2. إضافة المصادقة الاجتماعية (تسجيل الدخول باستخدام Google أو Facebook)
  3. إضافة تحقق CAPTCHA لمنع الروبوتات
  4. تحسين تجربة المستخدم على الأجهزة المحمولة
  5. إضافة رسائل خطأ أكثر تفصيلاً وتوجيهًا
