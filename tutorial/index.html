مرحباً بك في دورة تعلم تطوير الويب

أهلاً بك في دورتنا التعليمية حول أساسيات تطوير الويب. في هذه الدورة، سنتعلم كيفية إنشاء صفحات ويب بسيطة باستخدام HTML وتنسيقها باستخدام CSS وإضافة التفاعل باستخدام JavaScript.

ما هو HTML؟

HTML (لغة ترميز النص التشعبي) هي اللغة الأساسية لإنشاء صفحات الويب. تستخدم HTML العلامات لتحديد هيكل المحتوى على صفحة الويب.

مثال بسيط لصفحة HTML:

<!DOCTYPE html>
<html>
<head>
    <title>عنوان الصفحة</title>
</head>
<body>
    <h1>مرحباً بالعالم!</h1>
    <p>هذه فقرة نصية.</p>
</body>
</html>

العناصر الأساسية في HTML

  • <html> - يحدد بداية ونهاية مستند HTML
  • <head> - يحتوي على معلومات حول المستند (مثل العنوان والوصف)
  • <title> - يحدد عنوان الصفحة الذي يظهر في شريط عنوان المتصفح
  • <body> - يحتوي على المحتوى المرئي للصفحة
  • <h1> إلى <h6> - عناوين بأحجام مختلفة
  • <p> - فقرة نصية
  • <a> - رابط تشعبي
  • <img> - صورة

كيفية إنشاء صفحة HTML بسيطة

  1. افتح محرر نصوص بسيط مثل Notepad أو VS Code
  2. اكتب هيكل HTML الأساسي كما في المثال أعلاه
  3. احفظ الملف بامتداد .html (مثل: index.html)
  4. افتح الملف في متصفح الويب لرؤية النتيجة

تمرين عملي:

حاول إنشاء صفحة HTML بسيطة تحتوي على عنوان، فقرة نصية، وصورة. يمكنك استخدام المثال التالي كنقطة بداية:

<!DOCTYPE html>
<html lang="ar" dir="rtl">
<head>
    <meta charset="UTF-8">
    <title>صفحتي الأولى</title>
</head>
<body>
    <h1>مرحباً بالعالم!</h1>
    <p>هذه أول صفحة ويب أقوم بإنشائها.</p>
    <img src="https://example.com/image.jpg" alt="وصف الصورة">
</body>
</html>
