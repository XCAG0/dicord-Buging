الدرس الأول: أساسيات CSS

بعد أن تعلمنا كيفية إنشاء صفحة HTML بسيطة، سنتعلم الآن كيفية تنسيق هذه الصفحة باستخدام CSS (أوراق الأنماط المتتالية).

ما هو CSS؟

CSS هي لغة تستخدم لوصف كيفية عرض عناصر HTML. باستخدام CSS، يمكنك تغيير ألوان النص، وأحجام الخطوط، والهوامش، والتباعد، وترتيب العناصر، وغير ذلك الكثير.

كيفية إضافة CSS إلى صفحة HTML

هناك ثلاث طرق لإضافة CSS إلى صفحة HTML:

1. CSS داخلي (في عنصر style)

<head>
    <style>
        body {
            background-color: #f0f0f0;
            font-family: Arial, sans-serif;
        }
        h1 {
            color: blue;
        }
    </style>
</head>

2. CSS خارجي (ملف منفصل)

أولاً، قم بإنشاء ملف CSS منفصل (مثل style.css) ثم قم بربطه بصفحة HTML:

<head>
    <link rel="stylesheet" href="style.css">
</head>

3. CSS مضمن (في سمة style للعنصر)

<h1 style="color: blue; font-size: 24px;">عنوان بلون أزرق</h1>

قواعد CSS الأساسية

تتكون قاعدة CSS من جزأين رئيسيين:

  • المحدد (Selector): يحدد العنصر الذي سيتم تطبيق النمط عليه
  • التصريح (Declaration): يحدد كيفية تنسيق العنصر
selector {
    property: value;
    property: value;
}

أمثلة على المحددات:

  • p - يحدد جميع عناصر الفقرة
  • .class-name - يحدد العناصر ذات الفئة المحددة
  • #id-name - يحدد العنصر ذو المعرف المحدد
  • p.class-name - يحدد عناصر الفقرة ذات الفئة المحددة

خصائص CSS الشائعة

الألوان والخلفيات

body {
    color: #333333;              /* لون النص */
    background-color: #f5f5f5;   /* لون الخلفية */
}

النص والخطوط

p {
    font-family: Arial, sans-serif;  /* نوع الخط */
    font-size: 16px;                 /* حجم الخط */
    font-weight: bold;               /* سماكة الخط */
    text-align: center;              /* محاذاة النص */
}

الحدود والهوامش والتباعد

div {
    border: 1px solid #cccccc;   /* الحدود */
    margin: 10px;                /* الهوامش الخارجية */
    padding: 15px;               /* التباعد الداخلي */
}

مثال عملي

لنقم بتطبيق بعض أنماط CSS على صفحة HTML بسيطة:

<!DOCTYPE html>
<html lang="ar" dir="rtl">
<head>
    <meta charset="UTF-8">
    <title>مثال على CSS</title>
    <style>
        body {
            font-family: Arial, sans-serif;
            background-color: #f0f0f0;
            margin: 0;
            padding: 20px;
        }
        
        h1 {
            color: #8A2BE2;
            text-align: center;
            border-bottom: 2px solid #8A2BE2;
            padding-bottom: 10px;
        }
        
        p {
            line-height: 1.6;
            color: #333;
        }
        
        .highlight {
            background-color: #ffff99;
            padding: 5px;
            border-radius: 3px;
        }
        
        #main-content {
            max-width: 800px;
            margin: 0 auto;
            background-color: white;
            padding: 20px;
            border-radius: 5px;
            box-shadow: 0 2px 5px rgba(0,0,0,0.1);
        }
    </style>
</head>
<body>
    <div id="main-content">
        <h1>مرحباً بالعالم!</h1>
        <p>هذه فقرة عادية.</p>
        <p>وهذه فقرة تحتوي على <span class="highlight">نص مميز</span> بلون مختلف.</p>
    </div>
</body>
</html>

النتيجة:

ستظهر صفحة بخلفية رمادية فاتحة، وعنوان بنفسجي اللون مع خط أسفله، ومحتوى في مربع أبيض مع ظل خفيف. النص المميز سيظهر بخلفية صفراء فاتحة.

تمرين عملي:

حاول تعديل المثال السابق بإضافة المزيد من الأنماط مثل:

  • تغيير لون الخلفية
  • إضافة صورة خلفية
  • تغيير حجم الخط ونوعه
  • إضافة حدود وظلال إضافية
